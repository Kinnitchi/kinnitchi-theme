# Change Log

All notable changes to the "kinnitchi-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.1.2] - 2026-04-16

### Fixed (Neon Darkula)
- Fixed `editor.lineHighlightBorder` to match line highlight background (removed transparent blue tint)
- Added explicit `editorBracketMatch.border` color
- Changed bracket highlight foreground 1 from salmon/red to teal (`#80FFEA`) for better color balance
- Updated `editorBracketPairGuide.activeBackground1` and `background1` from salmon to teal
- Changed Keywords token color from pink (`#FF80BF`) to yellow (`#fede5d`)

## [1.1.1] - 2026-04-14

- Version bump for marketplace release

## [1.1.0] - 2026-04-14

### Changed (Neon Darkula)
- Adjusted color values across the Darkula theme for improved aesthetics
- Further refinements to syntax token colors and UI element contrast

## [1.0.3] - 2026-04-13

### Changed
- Renamed theme labels: `Neon Dark Theme` → `Neon Dark Modern`, `Darkrula` → `Neon Darkula`
- Renamed theme file `kinnitchi-theme-darkrula.json` → `kinnitchi-theme-darkula.json`
- Expanded Dark Modern theme with comprehensive UI and syntax token coverage
- Added more syntax token rules to Neon theme (functions, classes, operators, decorators, etc.)
- Updated color values in Neon Darkula and Neon themes for consistency and readability

## [1.0.2] - 2026-04-13

### Fixed
- Fixed Darkrula theme path in package.json
- Updated editor background for Dark Modern theme to `#1F1F1F`
- Updated Darkrula editor background to match Kinnitchi Neon (`#262A33`)

## [1.0.1] - 2026-04-13

### Added
- Added Darkrula theme variant

## [1.0.0] - Initial release

- Initial release with Neon and Neon Dark themes
